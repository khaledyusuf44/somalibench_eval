# Paper workspace -- SomaliBench Eval

This directory mirrors the SomaliWeb v1 paper setup: one self-contained LaTeX source, one BibTeX file, standard TeXLive packages only, and a small arXiv zip target.

```
paper/
|-- main.tex                 arXiv paper source
|-- refs.bib                 BibTeX references
|-- Makefile                 `make figs / pdf / zip / clean`
|-- HOWTO_SUBMIT.md          arXiv submission checklist
|-- figures/                 figure PDFs copied from ../data/figures
`-- arxiv-submission.zip     generated by `make zip`
```

## Build commands

Refresh figure PDFs from the public-safe analysis outputs:

```bash
make figs
```

Build the arXiv/Overleaf source package:

```bash
make zip
```

Build the PDF locally, if a LaTeX distribution is installed:

```bash
make pdf
```

## Safety boundary

The arXiv package includes only `main.tex`, `refs.bib`, and aggregate-result figures. It excludes raw model responses, classification JSONL files, the filled manual-review CSV, virtualenvs, logs, and secrets.

## Citation note

Aya-23-8B is described in the paper as a 23-language multilingual comparator, not as Somali-specific. The Aya 23 technical report / arXiv paper does not list Somali among its 23 covered languages.
